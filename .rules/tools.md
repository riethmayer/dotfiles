# Tool Preferences

Prefer these tools over defaults:
- `gh` over `git` for GitHub operations (PRs, issues, repos)
- `rg` (ripgrep) over `grep` - faster, respects .gitignore
- `fd` over `find` - simpler syntax, faster
- `bat` over `cat` - syntax highlighting
- `zoxide` (`z`) over `cd` - frecency-based jumps
- `lazygit` for complex git operations
- `fzf` for fuzzy finding files, history, branches
