# tmux Keybinds

Prefix: `C-a` (Ctrl+a)

## Splitting Panes

| Keys | Action |
|------|--------|
| `C-a s` | Split horizontal (panes stacked) |
| `C-a v` | Split vertical (side-by-side) |

## Pane Navigation (vim-style)

| Keys | Action |
|------|--------|
| `C-a h` | Move left |
| `C-a j` | Move down |
| `C-a k` | Move up |
| `C-a l` | Move right |

## Window Navigation

| Keys | Action |
|------|--------|
| `C-a H` | Previous window |
| `C-a L` | Next window |
| `C-a <space>` | Last window (toggle) |

## Windows

| Keys | Action |
|------|--------|
| `C-a c` | New window (current path) |
| `C-a C-c` | New window (home dir) |

## Session Management

| Keys | Action |
|------|--------|
| `C-a ^D` | Detach |
| `C-a o` | Session picker (sessionx + zoxide) |

## Plugins

| Keys | Action |
|------|--------|
| `C-a p` | Float toggle (floax) |
| `C-a I` | Install plugins (tpm) |
| `C-a U` | Update plugins (tpm) |

## Copy Mode (vi)

| Keys | Action |
|------|--------|
| `C-a [` | Enter copy mode |
| `v` | Begin selection |
| `y` | Copy and exit |

## Config

| Keys | Action |
|------|--------|
| `C-a r` | Reload config |
| `C-a C-r` | Reload config |
