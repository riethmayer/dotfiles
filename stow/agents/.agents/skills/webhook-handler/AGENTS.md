## Python Execution
- MUST use `uv` for Python workflows whenever a `uv` path exists.
- Use `uv run ...` for project scripts.
- Use `uv tool run ...` for Python CLI tools.

## CLI Commands
- Discover tasks: `mise tasks`
- Run task: `mise run <task>`
