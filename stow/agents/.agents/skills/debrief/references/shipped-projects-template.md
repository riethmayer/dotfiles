# Shipped Projects

Append-only index of shipped workstreams, newest first. Each line links to its
debrief. The `daily-obsidian` skill reads this file and renders the most recent
entries at the top of the standup section.

Line format (paths are relative to this file, which lives at the Journal root):

```
- YYYY-MM-DD · **<Project>** — [debrief](YYYY/MM-Month/YYYY-MM-DD/YYYY-MM-DD-Day-<slug>-debrief.html) · <one-clause summary>
```
